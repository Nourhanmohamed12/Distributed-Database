🌐 Distributed Database System

Comprehensive implementation of a Distributed Database System demonstrating data distribution, replication, and query processing across multiple nodes.

✨ Features
Feature	Description
🗂️ Data Distribution	Partitioning data across multiple nodes
🔁 Replication	Ensures fault tolerance and availability
🔍 Query Processing	Execute queries across distributed nodes
⚡ Concurrency Control	Handles simultaneous transactions
🔐 Data Consistency	Maintains integrity across replicas
🌍 Scalability	Easily extend system with new nodes
📊 Project Overview

This project simulates a distributed database environment, where data is stored across multiple databases instead of a single centralized system.

It demonstrates:

📦 Fragmentation (Horizontal / Vertical)
🔁 Replication Strategies
🔗 Distributed Query Execution
⚖️ Consistency vs Availability trade-offs
🛠️ System Architecture
